Documentation (soon)
-------------

The full documentation is at https://django-rest-localflavor.readthedocs.org.

Quickstart
----------

Install django-rest-localflavor::

    pip install django-rest-localflavor

Then use it in a project::

    INSTALLED_APPS = (
        # ...
        'rest_framework',
        'rest_localflavor',
    )
